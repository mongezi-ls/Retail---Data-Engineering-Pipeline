# Retail Sales Pipeline

## Overview

This project is a retail sales ETL pipeline built with Python. It uses the Medallion Architecture (Bronze, Silver, and Gold) to transform raw sales data into clean, business-ready data.

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Git
* GitHub
* Visual Studio Code

## Project Structure

* `data/` – Stores the project data.
* `src/` – Contains the Python source code.
* `docs/` – Project documentation.
* `diagrams/` – Architecture and workflow diagrams.
* `logs/` – Pipeline log files.
* `outputs/` – Reports, charts, and exported files.
* `tests/` – Test files.
